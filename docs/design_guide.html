Implementation Snippets

Get started quickly. Here are code snippets to integrate this color palette into your projects.

CSS Variables

Add these to your global CSS file for easy access to the palette throughout your application.

:root {
    --color-primary: #0a2342;
    --color-secondary: #2ca58d;
    --color-accent: #f0a500;
    --color-accent-light: #8db9d9;
    --color-bg: #f7f9fb;
    --color-border: #e1e6ea;
    --color-text-primary: #333333;
    --color-text-secondary: #a9b4c2;
}

Tailwind CSS Config

Extend your `tailwind.config.js` file to use these colors with Tailwind's utility classes.

module.exports = {
    theme: {
        extend: {
            colors: {
                'sdc-blue': '#0a2342',
                'sdc-teal': '#2ca58d',
                'sdc-gold': '#f0a500',
                'sdc-light-blue': '#8db9d9',
                'sdc-off-white': '#f7f9fb',
                'sdc-light-gray': '#e1e6ea',
                'sdc-med-gray': '#a9b4c2',
                'sdc-dark-gray': '#333333',
            },
        },
    },
    plugins: [],
}

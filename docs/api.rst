Triplestore Adapter Scaffolds
----------------------------

AxiusMEM provides a common interface and scaffolding for a wide range of RDF triplestores. Each adapter is a stub ready for community contribution and future extensibility. See the source code for details and to contribute an implementation for your preferred backend.
